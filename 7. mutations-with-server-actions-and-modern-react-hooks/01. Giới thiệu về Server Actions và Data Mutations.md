## Giới thiệu về Server Actions và Data Mutations

### Bối cảnh hiện tại

Cho đến thời điểm này, người dùng chưa thể thực hiện bất kỳ thao tác nào trên website. Trang web hiện tại chỉ đơn thuần hiển thị dữ liệu về các cabin (nhà gỗ), nhưng chưa có tính tương tác thực sự.

### Nội dung chính của section

Đây là section học về **Server Actions** - phương pháp để thay đổi dữ liệu (data mutation) trong mô hình [[React Server Components]].

#### Các tính năng sẽ được triển khai

Sử dụng [[Server Actions]] để xây dựng các tính năng quan trọng liên quan đến thay đổi dữ liệu:

- Đặt phòng cabin (reserving a cabin)
- Cập nhật đặt phòng (updating reservation)
- Xóa đặt phòng (deleting reservation)
- Cho phép người dùng cập nhật hồ sơ cá nhân (update profile)


### React Hooks mới

Để làm cho các tính năng thực sự tương tác và có hiệu suất tốt, section này sẽ giới thiệu các React hooks tiên tiến:

- `useFormStatus` - theo dõi trạng thái form
- `useTransition` - xử lý chuyển đổi trạng thái
- `useOptimistic` - cập nhật giao diện lạc quan (optimistic UI updates)


### Đánh giá

Section này mang lại nhiều kiến thức mới và quan trọng để hoàn thiện dự án.
