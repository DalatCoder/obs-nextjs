## Thiết Lập Tailwind CSS Trong Next.js

### Kích Hoạt Tailwind CSS

Tailwind CSS đã được cài đặt tự động khi chạy `create-next-app` vì đã chọn tùy chọn cài đặt Tailwind ngay từ đầu.

**Các file đã được tạo sẵn:**

- File `tailwind.config.js` trong thư mục root
- File `globals.css` trong thư mục styles với 3 dòng code Tailwind directives

**Cách kích hoạt:**

- Import file `globals.css` vào bất kỳ đâu trong ứng dụng
- Vị trí tốt nhất là trong `layout.js` vì nó bao bọc toàn bộ ứng dụng

**Ví dụ code:**

```javascript
// app/layout.js
import '@/app/_styles/globals.css'

export default function RootLayout({ children }) {
  return (
    <html>
      <body>{children}</body>
    </html>
  )
}
```


### Sử dụng Utility Classes

**Khái niệm:** Tailwind là một bộ sưu tập lớn các lớp tiện ích (utility classes) có thể kết hợp để tạo bất kỳ CSS nào.

**Cách viết styles:**

- Viết trực tiếp trong thuộc tính `className`
- Không cần tạo class riêng trong file CSS riêng biệt
- Tất cả styles nằm cùng element được áp dụng

**Ví dụ cơ bản:**

```javascript
<body className="bg-blue-800 text-gray-50 min-h-screen">
  {children}
</body>
```

**Giải thích các utility classes:**

- `bg-blue-800`: Background màu xanh dương độ đậm 800
- `text-gray-50`: Màu chữ xám rất nhạt (gần trắng)
- `min-h-screen`: Chiều cao tối thiểu bằng 100vh (100% viewport height)

**Xem CSS tương ứng:**

- Hover chuột lên class để xem CSS được áp dụng
- Ví dụ: `gap-4` tương ứng với `gap: 1rem` (16px)
- `gap-5` tương ứng với `gap: 1.25rem` (20px)


### Lợi Ích Của Tailwind

**Tính năng nổi bật:**

- Viết styles trực tiếp trong JSX, không cần file CSS riêng
- Hệ thống thiết kế (design system) tích hợp sẵn
- Các giá trị được định nghĩa trước hợp lý và đẹp mắt
- Phù hợp cho developers không giỏi thiết kế

**IntelliSense/Auto-completion:**

- Cần cài extension "Tailwind CSS IntelliSense"
- Hiển thị gợi ý khi gõ class names
- Hiển thị CSS tương ứng khi hover


### Tùy Chỉnh Theme

**Vấn đề:** Màu sắc mặc định của Tailwind không khớp với thương hiệu (brand) của dự án.

**Giải pháp:** Extend theme trong file `tailwind.config.js` với màu sắc tùy chỉnh.

**Cấu trúc file colors.json:**

```json
{
  "primary": {
    "50": "#f0f9ff",
    "100": "#e0f2fe",
    "200": "#bae6fd",
    "950": "#082f49"
  },
  "accent": {
    "50": "#fdf8f3",
    "500": "#f97316",
    "600": "#ea580c"
  }
}
```

**Cấu hình trong tailwind.config.js:**

```javascript
// tailwind.config.js
module.exports = {
  content: [
    "./app/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {
      colors: {
        primary: {
          50: '#f0f9ff',
          100: '#e0f2fe',
          200: '#bae6fd',
          // ... các shade khác
          950: '#082f49'
        },
        accent: {
          50: '#fdf8f3',
          500: '#f97316',
          600: '#ea580c'
        }
      }
    }
  }
}
```

**Sử dụng custom colors:**

```javascript
<body className="bg-primary-950 text-primary-50">
  {children}
</body>
```

- `bg-primary-950`: Background với màu primary độ đậm nhất
- `text-primary-50`: Text với màu primary rất nhạt
- Tạo contrast đẹp mắt cho giao diện


### Các Giải Pháp Styling Khác

**CSS Modules:**

- Được Next.js hỗ trợ sẵn (out of the box)
- Tạo scope CSS riêng cho từng component
- Giải pháp tốt, ổn định

**Styled-components:**

- CSS-in-JS solution
- Đã sử dụng trong phần đầu của Wild Oasis project
- Đang dần lỗi thời, ít phổ biến hơn

**Lý do chọn Tailwind:**

- Giải pháp hiện đại, dễ sử dụng
- Phổ biến trong cộng đồng Next.js
- Phù hợp với xu hướng phát triển web hiện nay


### Cài Đặt Icon Library

**Heroicons:** Thư viện icon từ team phát triển Tailwind CSS.

**Cài đặt:**

```bash
npm install @heroicons/react
```

**Sử dụng:**

```javascript
import { HomeIcon, UserIcon } from '@heroicons/react/24/solid'

function Navigation() {
  return (
    <nav>
      <HomeIcon className="w-6 h-6" />
      <UserIcon className="w-6 h-6" />
    </nav>
  )
}
```


### Xử Lý Lỗi Thường Gặp

**Lỗi trong globals.css:**

- Thường xuất hiện khi chỉnh sửa file config
- Restart dev server thường fix được vấn đề
- Comment/uncomment code có thể giúp tìm lỗi

**Thiếu IntelliSense:**

- Cài đặt extension "Tailwind CSS IntelliSense"
- Restart VS Code sau khi cài
- Kiểm tra file `tailwind.config.js` đã đúng cấu trúc

**Import không hoạt động:**

- Đảm bảo đường dẫn đến `globals.css` chính xác
- Sử dụng import alias `@/app` cho đường dẫn sạch hơn
- Kiểm tra file có tồn tại trong thư mục `_styles`


### Ghi Chú Thêm

- Tailwind đã được kích hoạt khi import `globals.css` vào layout
- Tất cả component con sẽ có thể sử dụng Tailwind classes
- Custom colors giúp giữ nhất quán thương hiệu trong toàn bộ ứng dụng
- Heroicons cung cấp icons đẹp, tích hợp tốt với Tailwind
- Phần tiếp theo sẽ thêm metadata và favicon cho các pages
