## Xây Dựng Trang Cabins và About

### Trang Cabins Overview

Mở file `page.js` trong thư mục `cabins` và thay thế nội dung bằng code từ starter pages. Trang này hiển thị danh sách các cabin dưới dạng cards (thẻ).

**Cấu trúc chính:**

- Tiêu đề được styled với texture đẹp mắt
- Mảng cabins (hiện tại để trống, sẽ cập nhật sau)
- Component CabinCard để render từng cabin

**Import CabinCard component:**

```javascript
import CabinCard from '@/components/CabinCard'
```

Khi có dữ liệu, mỗi cabin sẽ được render thành một card. Người dùng click vào card sẽ chuyển đến trang chi tiết cabin (sẽ implement sau).

### Trang About với Responsive Images

Trang About chứa nhiều hình ảnh cần tối ưu hóa. Ban đầu, hình ảnh có kích thước rất lớn (1.1MB, 2000 pixels chiều rộng) dù không cần thiết.

**Vấn đề:** Khi replace `<img>` bằng Next.js Image component, cần giữ tính responsive - hình ảnh phải co giãn theo kích thước column trong grid layout.

### Hai Phương Pháp Tối Ưu Responsive Image

**Phương pháp 1: Static Import (khuyên dùng khi có thể)**

Khi có thể import hình ảnh trực tiếp từ folder:

```javascript
import Image from 'next/image'
import imageOne from '@/public/about-1.jpeg'

<Image 
  src={imageOne}
  placeholder="blur"
  quality={80}
  alt="Description"
/>
```

**Lợi ích:**

- Không cần chỉ định width/height
- Tự động responsive nhờ Tailwind CSS `max-width: 100%`
- Có thể dùng `placeholder="blur"`
- Có thể điều chỉnh `quality`
- Next.js tự động phân tích và tối ưu

**Phương pháp 2: Container với Fill Property**

Khi nhận URL hình ảnh từ database hoặc API (không thể static import):

```javascript
<div className="relative aspect-square">
  <Image 
    src="/about-1.jpeg"
    fill
    className="object-cover"
    alt="Description"
  />
</div>
```

**Kỹ thuật quan trọng:**

1. **Tạo container div:** Bọc Image component trong một div
2. **Thêm `position: relative`:** Dùng class `relative` trên container
3. **Chỉ định kích thước container:** Sử dụng `aspect-ratio` thay vì width/height cố định
4. **Sử dụng `fill` prop:** Giúp hình ảnh lấp đầy container
5. **Thêm `object-cover`:** Giữ tỷ lệ khung hình và fill toàn bộ container

### Aspect Ratio Options

Tailwind CSS cung cấp các utility classes cho aspect ratio:

- `aspect-square` - tỷ lệ 1:1
- `aspect-video` - tỷ lệ 16:9
- `aspect-[4/3]` - tỷ lệ tùy chỉnh

**Ví dụ sử dụng aspect-video:**

```javascript
<div className="relative aspect-video">
  <Image 
    src="/image.jpg"
    fill
    className="object-cover"
    alt="Description"
  />
</div>
```


### So Sánh Hai Phương Pháp

**Static Import:**

- Phù hợp với hình ảnh có sẵn trong project
- Hỗ trợ `placeholder="blur"`
- Hỗ trợ điều chỉnh `quality`
- Tự động responsive với Tailwind
- Next.js phân tích trước kích thước

**Container với Fill:**

- Bắt buộc khi URL động từ database/API
- Không hỗ trợ `placeholder="blur"`
- Không hỗ trợ điều chỉnh `quality`
- Cần setup container thủ công
- Linh hoạt hơn với layout phức tạp


### Grid Layout trong Trang About

Trang About sử dụng CSS Grid với mỗi hàng là một grid container:

```javascript
<div className="grid grid-cols-5 gap-x-24">
  <div className="col-span-3">
    {/* Nội dung text */}
  </div>
  <div className="relative col-span-2 aspect-square">
    <Image 
      src="/about-1.jpeg"
      fill
      className="object-cover"
      alt="Description"
    />
  </div>
</div>
```

Hình ảnh chiếm 2 cột trong grid 5 cột, tự động co giãn theo chiều rộng cột.

### Layout Shift và SEO

Việc ngăn chặn layout shift (dịch chuyển bố cục) khi tải hình ảnh là metric quan trọng cho Google SEO. Next.js Image component tự động xử lý điều này thông qua:

- Yêu cầu dimensions hoặc sử dụng `fill` với container có kích thước xác định
- Placeholder blur để người dùng thấy "có gì đó" đang tải
- Không để background trống đột ngột xuất hiện nội dung


### Code Mẫu Hoàn Chỉnh

**Với Static Import:**

```javascript
import Image from 'next/image'
import aboutOne from '@/public/about-1.jpeg'

<Image 
  src={aboutOne}
  placeholder="blur"
  quality={85}
  className="w-full h-auto"
  alt="About image 1"
/>
```

**Với Dynamic URL:**

```javascript
<div className="relative aspect-square">
  <Image 
    src={imageUrl}
    fill
    className="object-cover"
    alt="Dynamic image"
  />
</div>
```


### Ghi Chú Thêm

- API của Image component khá phức tạp ban đầu nhưng dễ hiểu sau vài lần thực hành
- Aspect ratio là CSS property quan trọng nhưng thường bị bỏ qua
- Kỹ thuật container với `fill` property rất hữu ích và sẽ dùng lại nhiều lần trong project
- Khi production với web server thực, placeholder blur sẽ hiển thị lâu hơn và cải thiện trải nghiệm người dùng
- Hình ảnh được tối ưu từ 1.1MB xuống chỉ vài chục KB, tiết kiệm bandwidth đáng kể
