## Xây Dựng Trang Chủ (Homepage)

### Thiết Lập Nội Dung Trang

Để xây dựng trang chủ, mở file `page.js` trong thư mục root của app. Thay thế nội dung starter code bằng JSX từ file mẫu trong thư mục `pages/homepage`.

**Các thành phần chính:**

- Thẻ `<h1>` cho tiêu đề
- Button chuyển hướng đến trang cabins (cần chuyển thành Link component)
- Hình ảnh nền lớn cần được tối ưu


### Tối Ưu Hóa Hình Ảnh Nền

**Import Image Component:**

```javascript
import Image from 'next/image'
import bg from '@/public/bg.png'
```

Next.js sẽ cảnh báo nếu sử dụng thẻ `<img>` thông thường và khuyến nghị dùng Image component để tự động tối ưu hóa.

### Sử Dụng Thuộc Tính Fill

Thay vì chỉ định `width` và `height` cố định, sử dụng thuộc tính `fill` để tạo responsive image (hình ảnh co giãn theo màn hình):

```javascript
<Image 
  src={bg}
  fill
  alt="Background"
/>
```

**Lợi ích:** Hình ảnh sẽ tự động chiếm toàn bộ chiều rộng màn hình và điều chỉnh kích thước phù hợp.

### Các Thuộc Tính CSS Quan Trọng

**Object Fit - Cover:**

```javascript
<Image 
  src={bg}
  fill
  className="object-cover"
  alt="Background"
/>
```

Tương đương CSS: `object-fit: cover` - giúp hình ảnh giữ tỷ lệ khung hình (aspect ratio) và lấp đầy toàn bộ container, tương tự như `background-size: cover`.

**Object Position - Top:**

```javascript
<Image 
  src={bg}
  fill
  className="object-cover object-top"
  alt="Background"
/>
```

Tương đương CSS: `object-position: top` - giữ hình ảnh cố định từ phía trên khi viewport thay đổi kích thước, tránh hiện tượng hình ảnh "nhảy" khi resize.

### Hiệu Ứng Placeholder Blur

Thêm hiệu ứng mờ (blur) khi hình ảnh đang tải - chỉ hoạt động với static import:

```javascript
<Image 
  src={bg}
  fill
  placeholder="blur"
  quality={80}
  className="object-cover object-top"
  alt="Background"
/>
```

**Kết quả:** Hình ảnh sẽ hiển thị dạng mờ trong quá trình tải, sau đó hiện rõ nét khi tải xong - tạo trải nghiệm người dùng mượt mà hơn.

### Điều Chỉnh Quality

Vì đây là hình nền, không cần chất lượng quá cao. Sử dụng `quality={80}` thay vì 100% để giảm kích thước file mà vẫn giữ chất lượng tốt.

**So sánh:** Hình ảnh gốc khoảng 7MB, sau khi tối ưu chỉ còn khoảng 45KB - tiết kiệm bandwidth đáng kể.

### Xử Lý Position Warning

Khi sử dụng `fill`, cần có parent element với `position: relative`. Thêm class vào `<body>` trong layout:

```javascript
<body className="relative">
  {children}
</body>
```

**Lưu ý:** Nếu thêm `position: relative` vào element con, hình ảnh sẽ bị giới hạn trong container đó thay vì toàn trang.

### Tối Ưu Responsive Tự Động

Next.js tự động phục vụ hình ảnh với kích thước phù hợp dựa trên viewport:

- Màn hình nhỏ: hình ảnh 1000x667 pixels
- Màn hình lớn: hình ảnh 3000x2000 pixels

Điều này xảy ra on-demand (theo yêu cầu) khi trang được tải, giúp tối ưu hiệu suất mà không tốn công xử lý thủ công.

### Code Hoàn Chỉnh

```javascript
<Image 
  src={bg}
  fill
  placeholder="blur"
  quality={80}
  className="object-cover object-top"
  alt="Mountains and forests with two cabins"
/>
```


### Ghi Chú Thêm

- Không thể sử dụng hình ảnh như `background-image` trong CSS khi dùng Image component
- Thuộc tính `fill` kết hợp với `object-cover` và `object-top` tạo hiệu ứng tương tự background image
- Hình ảnh được sinh bởi AI (Mid Journey) tạo thiết kế đẹp mắt cho portfolio
- Kích thước file được tối ưu tự động, tiết kiệm bandwidth cho người dùng
