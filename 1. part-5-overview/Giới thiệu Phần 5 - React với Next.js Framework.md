## Giới thiệu Phần 5: React với Next.js Framework

### Tổng quan

Phần 5 của khóa học tập trung vào việc đưa [[React]] lên server và tận dụng các tính năng tiên tiến nhất thông qua framework [[Next.js]] - một trong những framework mạnh mẽ và phổ biến nhất hiện nay.

### Mục tiêu học tập

Trong các phần tiếp theo, khóa học sẽ hướng dẫn xây dựng một website hướng đến khách hàng (customer-facing website) cho dự án **The Wild Oasis** (đã làm việc ở Phần 4), đồng thời khám phá:

- [[Server Side Rendering]] (SSR) - kỹ thuật render phía server
- [[File-based Routing]] - cơ chế định tuyến dựa trên cấu trúc file
- [[React Suspense]] - tính năng xử lý bất đồng bộ tiên tiến
- [[Server Components]] - components render trên server


### Tại sao Next.js quan trọng?

- Các công nghệ này đang trở thành tiêu chuẩn để xây dựng ứng dụng [[React]] hiện đại và có hiệu suất cao
- Ngày nay, việc thành thạo [[Next.js]] gần như trở thành yêu cầu bắt buộc đối với các [[React Developer]]
- Đây là lý do Phần 5 được thêm vào để biến khóa học thành "Ultimate React Course" toàn diện hơn


### Dự án thực hành

Xây dựng website khách hàng cho **The Wild Oasis** - ứng dụng các kiến thức:

- Tích hợp [[React]] với server-side
- Triển khai [[Next.js]] features
- Áp dụng các pattern hiện đại
